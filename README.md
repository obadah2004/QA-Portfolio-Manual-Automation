# QA Portfolio – Manual & Automation Testing

## 📌 Overview

This repository showcases my QA skills in both **Manual Testing** and **Test Automation**.
The project focuses on testing a real-world e-commerce application to ensure quality, reliability, and usability.

---

## 🧪 Project Scope

Application tested: Sauce Demo (E-commerce web application)

### Covered Modules:

* Login Functionality
* Product Listing
* Cart Management
* Checkout Process

## 🛠️ Tools & Technologies

* Manual Testing (Test Cases & Bug Reports)
* Selenium WebDriver (C#)
* NUnit (Test Framework)
* API Testing using Postman
* Git & GitHub for version control
